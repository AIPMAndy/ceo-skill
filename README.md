# ceo-skill

A CEO decision-making skill for agent environments.

This repository contains:

- `SKILL.md` — the main skill definition
- `references/` — decision frameworks, debiasing, stakeholder, and war-gaming references
- `scripts/analysis_tools.py` — lightweight modeling helpers for Monte Carlo, ICE scoring, decision matrices, EV, NPV, and IRR
- `evals/evals.json` — sample evaluation prompts

## What this skill is for

Use it for executive-level decision support, including:

- strategy and trade-off analysis
- resource allocation and prioritization
- crisis response
- stakeholder navigation
- competitive war gaming
- debiasing and assumption checks

## Notes

The main skill instructions live in `SKILL.md`.
This repo is intentionally lightweight so it can be adapted to different agent runtimes.
